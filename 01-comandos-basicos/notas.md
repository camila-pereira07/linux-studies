# COMANDOS BÁSICO DE LINUX:

## Navegação

- ls (lista de arquivos);
- cd (muda diretório);
- pwd (mostra diretório atual).

## Manipulação

- mkdir (criar pastas/diretórios);
- touch (criar arquivos);
- rm (remove arquivos);
- cp (copia arquivos);
- mv (move arquivos).

## Prática

- Usei esse comandos no terminal do Ubuntu para criar e organizar arquivos.
