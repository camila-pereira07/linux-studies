- Gerenciamento de arquivos;
- Bash scripting;
- Análise básica de logs;
- Fundamentos de administração de sistemas.

## Conteúdo

- Comandos básico do Linux;
- Manipulação de arquivos;
- Indrodução do Bash;
- Exercícios práticos em ambiente virtual.
